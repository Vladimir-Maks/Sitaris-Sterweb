Паттерны представлены на репозитории моего курсового проекта (https://github.com/Vladimir-Maks/AlisaWorld)

Там клиентская часть имеет в себе несколько патернов:
1.Синглтон, само приожение впринципе является синглтоном, которое создает окно с графикой и рисует в нем в то же время обрабатывая инпут пользователя
2.Хранитель, окно инициализируется с помощью класса конфигуратора который оно принимает в виде параметра в конструкторе
